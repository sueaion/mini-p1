# 📘 일만 시간의 법칙 (The 10,000 Hours Rule)

> “연습은 어제의 당신보다 당신을 더 낫게 만든다.”

이 프로젝트는 **1만 시간의 법칙**을 주제로,  
사용자가 하루 훈련 시간을 입력하면  
전문가가 되기까지 필요한 훈련 기간을 직관적으로 보여주는  
**모바일 퍼스트 랜딩 페이지**입니다.

---

## 🔗 프로젝트 개요

- 정적 웹 페이지 (HTML / CSS)
- 모바일 화면 기준 UI 설계
- 사용자의 입력 흐름에 맞춘 화면 구성
- 결과 안내 및 응원 메시지 모달 UI 구현

---

## 🎯 제작 목적

- HTML 시멘틱 태그를 올바르게 사용한다.
- CSS를 활용해 모바일 환경에 적합한 UI를 구현한다.
- 사용자 입력 → 결과 확인 → 행동 유도의 흐름을 설계한다.
- 웹 접근성을 고려한 마크업을 연습한다.

---

## 🛠 사용 기술

- HTML5  
- CSS3  
- Web Font (Noonnu, Google Fonts)

## 🧩 주요 화면 구성

### 1️⃣ 인트로 영역
- 1만 시간의 법칙에 대한 간단한 설명 제공
- 인용 이미지와 텍스트를 활용한 시각적 강조

### 2️⃣ 계산 영역
- 목표 분야 입력
- 하루 훈련 시간 입력
- 버튼 클릭 시 결과 영역 노출

### 3️⃣ 결과 영역
- 강조된 숫자와 텍스트로 훈련 기간 안내
- 사용자 동기부여를 위한 메시지 구성

### 4️⃣ 모달 UI
- “훈련하러 가기” 버튼 클릭 시 응원 메시지 표시
- 단일 행동을 유도하는 CTA 버튼 제공

---

## ♿ 웹 접근성 & 마크업 고려

- `header`, `main`, `footer` 등 시멘틱 태그 사용
- 이미지에 의미 있는 `alt` 텍스트 작성
- 텍스트 흐름을 고려한 문단 구성
- IR 기법을 활용한 시각 요소 처리

---

## 💡 느낀 점

이번 프로젝트를 통해  
단순한 화면 구현보다 **사용자 흐름과 가독성**이 중요하다는 점을 배웠고,  
모바일 기준 UI 설계와 시멘틱 마크업의 필요성을 이해하게 되었습니다.
